Eir Scheduling Solutions:

This project is a Full-Stack application. The server has API's that the front end can use to obtain the data that is populated on the calendar. The software is a simple scheduling application geared towards doctors offices. with it you can:

  • Securely Register and login
  • Create a New Patient
  • Create New Appointments that are linked to patients on the server
  • All Calendar events are clickable and draggable
  • Data persists and is stored securely in a database

Back End: Node JS | Packages: Express, Cors, BCrypt

Front End: JavaScript HTML CSS | Packages: Axios,
